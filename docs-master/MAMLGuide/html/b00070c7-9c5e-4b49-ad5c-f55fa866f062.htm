Known Issues and Bugs

This topic covers known issues and bugs with Sandcastle and conceptual content. Currently, the VS2005 presentation style is the best choice for conceptual content as it contains no known issues beyond those noted in the General Issues section. The list does not include issues fixed by the noted patch.

General Issues

These are the currently known issues with Sandcastle in general when used to produce conceptual content regardless of presentation style.

  • There does not appear to be any formatting for several of the Whitepaper topic elements.
  • Several elements of the XML Reference topic type do not render correctly if at all.
  • Sandcastle's XSL transformations remove inter-element whitespace. As such, if you place two MAML elements one after the other and require a separating space between them, use a non-breaking space entity (&#160;). See the replaceable topic for an example.
Hana Presentation Style Issues

These are the currently known issues with the Hana presentation style.

  • MS Help Viewer output is not supported as this style has been deprecated.
  • The language filter is present but only toggles syntax and code snippet visibility. It will not toggle the visibility on code examples.
 Note
This presentation style has been deprecated and is no longer supported by Microsoft.
Prototype Presentation Style Issues

These are the currently known issues with the Prototype presentation style.

  • MS Help Viewer output is not supported as this style has been deprecated.
  • The language filter is present but only toggles syntax and code snippet visibility. It will not toggle the visibility on code examples.
 Note
This presentation style has been deprecated and is no longer supported by Microsoft.
