CVT0005: Error reading project from '[file]'

Error CVT0005: Error reading project from '[file]' (last property = [property]): [error]

Cause

This is a generic catch-all error that occurs if there is a problem converting a project file.

Solution

Since this is a catch-all error, refer to the property name and error description for information about the cause of the error and how you might be able to correct it (i.e. bad value format, unrecognized value, etc.). If you are unable to solve the problem based on the reported error, open a work item in the Sandcastle Help File Builder project at CodePlex containing the details from the error message. Attach the affected source project as an example.
