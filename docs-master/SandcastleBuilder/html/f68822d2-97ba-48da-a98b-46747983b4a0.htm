Project File Converters

A set of project converters is supplied that can convert help projects in various other documentation tool formats into the help file builder project file format. To convert a project from the standalone GUI, select the File | New Project from Other Format menu option. If using Visual Studio, select File | New | Project and use the New Project from Other Format project template.

Available Project Converters

All project converters work in a similar fashion. Each requires the name of a project file and a destination folder in which to place the converted project and its related files. The destination folder must be different from the source folder that contains the project file being converted. If it does not exist, it will be created.
