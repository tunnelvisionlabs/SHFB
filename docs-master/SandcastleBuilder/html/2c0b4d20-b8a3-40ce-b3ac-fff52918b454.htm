Error PRJ0003: Error reading project from '[file]': [reason]

Cause

This is a generic catch-all error that occurs when there is a problem attempting to load a help file builder project. If you have been modifying the project file by hand, it could indicate that you have introduced an error into the help file properties.

Solution

Since this is a catch-all error, refer to the reason description for information about the cause of the error and how you might be able to correct it (i.e. file not found, permission issues, a corrupt file, etc.).

If you have been modifying the help file project by hand rather than by using the help file builder standalone GUI, back out your changes and try to reload the project. If you have not made any such modifications, open a work item in the Sandcastle Help File Builder project at CodePlex containing the details from the error message. Attach the affected project as an example.
