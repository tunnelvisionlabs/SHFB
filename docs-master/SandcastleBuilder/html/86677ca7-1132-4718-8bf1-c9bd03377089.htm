Version 1.5.0.1

Version 1.5.0.1 was released on June 26th, 2007.

Changes in This Release
  • Fixed a bug that caused "Missing namespace summary" messages and failure to exclude namespaces when using the VS2005 or HANA presentation style.
  • Fixed a bug in the code colorizer that caused it to fail to match C# and J# language IDs to the correct rule set.
  • Replaced the hard-coded path in the table of contents generation script files with the correct replacement tag to match the other commands.
