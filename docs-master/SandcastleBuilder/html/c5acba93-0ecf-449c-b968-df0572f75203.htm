Error PRJ0004: An attempt was made to set a read-only or ignored property: [property] Value: [value]

Cause

This is an internal error that indicates a problem with the project system or the project file. If you have been modifying the project file by hand, it could indicate that you have introduced an error into the help file properties.

Solution

If you have been modifying the help file project by hand rather than by using the help file builder standalone GUI, back out your changes and try to reload the project. If you have not made any such modifications, open a work item in the Sandcastle Help File Builder project at CodePlex containing the details from the error message. Attach the affected project as an example.
