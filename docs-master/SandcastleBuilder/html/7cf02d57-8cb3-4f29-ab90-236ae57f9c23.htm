SHE0003: Unexpected error applying command line option

Error SHE0003: Unexpected error applying command line option '[option]': [error]

Cause

This is a generic catch-all error that occurs if a problem is encountered while trying to parse the value of a command line option.

Solution

Since this is a catch-all error, refer to the error description for information about the cause of the error and how you might be able to correct it (i.e. bad option value, invalid format, etc.). If you are unable to solve the problem based on the reported error, open a work item in the Sandcastle Help File Builder project at CodePlex containing the details from the error message. Attach the affected project as an example.
