Warning GID0002: No comments found for cref '[cref]' on member '[id]'

Cause

This warning will appear of an <inheritDoc /> element specified a cref attribute but the resulting XPath query resulted in no matches.

Solution

Check the cref attribute to be sure that it is correct. In addition, check the comments files to be sure that it contains the elements that are to be selected by the XPath query.
