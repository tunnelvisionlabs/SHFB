SHE0004: The key '[key]' used for '[file]' is already in use

Warning SHE0004: The key '[key]' used for '[file]' is already in use by '[otherFile]'. '[file]' will be ignored.

Cause

This warning occurs if a duplicate topic filename is encountered while parsing files for titles and keywords. This can occur if HTML files are added to a project as additional content. If an identically named file occurs in two separate folders, only the first file will be included in the table of contents.

Solution

To correct the problem, give the duplicate file a unique name in the project. If the file is referenced in a site map file, edit the site map file to use the new name as well.
