Visual Studio Integration Package

An extension package is available that adds project management support to Visual Studio 2010 and later so that you can create, manage, and build help file builder projects from within Visual Studio just as you do for all other project types. When used, Visual Studio takes the place of the standalone GUI and provides all of the same features as well as the extra ones available within it such as source control, IntelliSense within the file editors, etc.
