Error CVT0006: Error converting content layout file ([file]): [error]

Cause

This is a generic catch-all error that occurs if there is a problem converting the conceptual content layout file.

Solution

Since this is a catch-all error, refer to the error description for information about the cause of the error and how you might be able to correct it (i.e. file not found, permissions error, etc.). If you are unable to solve the problem based on the reported error, open a work item in the Sandcastle Help File Builder project at CodePlex containing the details from the error message. Attach the affected source project and the indicated content layout file as an example.
