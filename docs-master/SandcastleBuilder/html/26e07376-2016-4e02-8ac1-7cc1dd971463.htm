Error ACP0001: The Additional Content Only plug-in requires a conceptual content layout file, a site map file, or content items in the project

Cause

This error occurs if the Additional Content Only plug-in cannot find at least one content layout file, site map file, or a file item with a build action of Content.

Solution

In order to produce a help file containing no API content, there must be at least one content layout file, a site map file, or some HTML content file items that can be used to produce non-API help topics that will appear in the help file. Add at least one required item and any additional files and perform the build again.
