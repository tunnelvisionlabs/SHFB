This namespace contains various classes used to convert MAML topics to WPF flow documents for previewing.
SandcastleBuilder.WPF.UserControls
This namespace contains a set of WPF user controls used by both the standalone GUI and the Visual Studio package to provide a consistent set of features and functionality to the file editors and tool windows used in both.
