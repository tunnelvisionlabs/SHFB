Version History

 Note

If you are coming to the latest version from version 1.7.0.0 or earlier be sure to read the Version 1.8.0.0 (Alpha), Version 1.8.0.1 (Beta 1), Version 1.8.0.2 (Beta 2), and Version 1.8.0.3 release notes as a significant number of changes occurred to the project file format and the standalone GUI across those releases. Any existing projects will need to be converted to the new project file format before they can be used with the new version.
