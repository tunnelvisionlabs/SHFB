SandcastleBuilder.Components Namespaces
This namespace contains BuildAssembler related namespaces.
Namespaces
NamespaceDescription
SandcastleBuilder.Components
This namespace contains custom build components used by the Sandcastle BuildAssembler tool. The components are used to add additional features to the help topics such as a code colorization, alternate framework data caching implementations, etc.
SandcastleBuilder.Components.Commands
This namespace contains command classes used by the BuildAssembler derived CopyFromIndexComponent classes.
SandcastleBuilder.Components.Targets
This namespace contains target data classes used by the BuildAssembler derived ResolveReferenceLinksComponent classes.
