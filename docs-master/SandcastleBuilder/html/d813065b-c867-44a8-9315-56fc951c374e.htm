PRJ0002: The selected file is for a more recent version

Error PRJ0002: The selected file is for a more recent version of the help file builder. Please upgrade your copy to load the file.

Cause

As indicated, the selected project file is for a more recent version of the help file builder. It contains project options that cannot be handled by your current release.

Solution

Upgrade to the latest release of Sandcastle and the Sandcastle Help File Builder.
