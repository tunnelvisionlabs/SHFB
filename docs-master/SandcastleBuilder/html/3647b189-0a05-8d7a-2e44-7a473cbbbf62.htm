SandcastleBuilder.InheritedDocumentation Namespace
This namespace contains documentation for the console mode application used to generate inherited documentation. The application supports running as an MSBuild task.
Classes
  ClassDescription
GenerateInheritedDocs
This class represents the tool that scans XML comments files for <inheritdoc /> tags and produces a new XML comments file containing the inherited documentation for use by Sandcastle.
ReflectionFiles
This is used to load one or more reflection data files used to look up API information
