Error CVT0002: An attempt was made to set an unknown or read-only property: [property] Value: [value]

Cause

This is an internal error that indicates a problem with the project system or the project file.

Solution

Open a work item in the Sandcastle Help File Builder project at CodePlex containing the details from the error message. Attach the affected source project as an example.
