B2X0001: A help compiler path must be specified

Error B2X0001: A help compiler path must be specified

Cause

This error occurs if the HelpCompilerFolder property on the Build2xHelpFile build task has not been set or is set to an empty string.

Solution

Specify a valid path for the HelpCompilerFolder property on the Build2xHelpFile build task in the project file.
