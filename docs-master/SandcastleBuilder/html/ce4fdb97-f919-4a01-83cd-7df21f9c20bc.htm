Error ARL0005: Unable to locate Resolve Reference Links component in [configFile]

Cause

The Resolve Reference Links component could not be located in the indicated configuration file.

Solution

This most likely indicates an internal error in the plug-in. Open a work item in the Sandcastle Help File Builder project at CodePlex containing the details from the error message and attach a small example project with any related files that can be used to duplicate the problem.
