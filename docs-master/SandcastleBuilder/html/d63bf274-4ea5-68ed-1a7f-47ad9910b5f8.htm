SandcastleBuilder.Utils Namespaces
This namespace contains all of the other utility and build engine related namespaces.
Namespaces
NamespaceDescription
SandcastleBuilder.Utils
This namespace contains various utility classes and the help file builder project classes.
SandcastleBuilder.Utils.BuildComponent
This namespace contains the classes used to load and manage third-party build components that can be used with the help file builder and Sandcastle tools (plug-ins, build components, syntax generators, and presentation styles).
SandcastleBuilder.Utils.BuildEngine
This namespace contains the classes related to the build engine.
SandcastleBuilder.Utils.ConceptualContent
This namespace contains classes related to conceptual content items.
SandcastleBuilder.Utils.Controls
This namespace contains various Windows Forms control classes.
SandcastleBuilder.Utils.Conversion
This namespace contains classes used to convert projects in other formats to the latest MSBuild project format used by the help file builder.
SandcastleBuilder.Utils.Design
This namespace contains various designer classes.
SandcastleBuilder.Utils.Gac
This namespace contains a couple of classes used to interact with the GAC and to load assemblies in an alternate application domain.
SandcastleBuilder.Utils.InheritedDocumentation
This namespace contains classes related to indexing comments files used when generating inherited documentation.
SandcastleBuilder.Utils.MSBuild
This namespace contains MSBuild related classes such as tasks and project file wrappers.
SandcastleBuilder.Utils.XPath
This namespace contains some classes that extend XPath queries with some custom functions used in the API filter editor.
