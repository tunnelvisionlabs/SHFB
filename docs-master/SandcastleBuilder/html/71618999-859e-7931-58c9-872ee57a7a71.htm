SandcastleBuilder Namespaces

This namespace contains all of the other Sandcastle Help File Builder related namespaces.
Namespaces
NamespaceDescription
SandcastleBuilder.Components
This namespace contains BuildAssembler related namespaces.
SandcastleBuilder.Gui
This namespace contains the classes for the standalone GUI help file builder and project manager.
SandcastleBuilder.Gui.ContentEditors
This namespace contains content editors used to edit the conceptual content items in the standalone GUI.
SandcastleBuilder.HtmlExtract
This namespace contains documentation for the console mode application used to extract title and keyword information from HTML files for use in creating the CHM table of contents and keyword index files. The application supports running as an MSBuild task.
SandcastleBuilder.InheritedDocumentation
This namespace contains documentation for the console mode application used to generate inherited documentation. The application supports running as an MSBuild task.
SandcastleBuilder.MicrosoftHelpViewer
This namespace contains classes related to the Microsoft Help Viewer help file format and the installation tool.
SandcastleBuilder.Package.PropertyPages
This namespace contains property pages for the standalone GUI and the Visual Studio package.
SandcastleBuilder.PlugIns
This namespace contains the plug-ins supplied with the Sandcastle Help File Builder. These can be used in a project to extend or alter the build process in various ways such as supporting build completion notification, deployment, and generating documentation with AjaxDoc.
SandcastleBuilder.Utils
This namespace contains all of the other utility and build engine related namespaces.
SandcastleBuilder.WPF
This namespace contains the WPF related namespaces
