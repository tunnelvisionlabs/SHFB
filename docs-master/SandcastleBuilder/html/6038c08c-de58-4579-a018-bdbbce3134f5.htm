GID0007: Ignoring non-XML comments file

Warning GID0007: Ignoring non-XML comments file

Cause

This warning occurs if a filename is encountered while indexing comments files that does not have a .xml extension.

Solution

If the file is not an XML comments file, the warning can be ignored. If it is intended to be a n XML comments file, rename it to use a .xml extension.
