Website Category Properties

The Website category properties control the features of the website output.

  • Website SDK link type (WebsiteSdkLinkType)
Website SDK link type (WebsiteSdkLinkType)

This property is used to define the type of links used to reference other help topics referring to framework (SDK) help topics. It can be set to None which renders non-clickable links or MSDN (the default) which renders links to the online MSDN help content for the topic. When set to MSDN, the Sdk Link Target property is used to define where the MSDN links will open in the browser.
