Property Value

Type: String
When set, if the path is not rooted (a relative path), IsFixedPath is set to false. If rooted (an absolute path), it is not changed. This property always returns a fully qualified path but without any environment variable expansions and terminated with a trailing backslash if needed.

If set to a null or empty string, the folder path is cleared and is considered to be undefined.
