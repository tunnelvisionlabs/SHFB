SandcastleBuilder.WPF.UserControls Namespace

This namespace contains a set of WPF user controls used by both the standalone GUI and the Visual Studio package to provide a consistent set of features and functionality to the file editors and tool windows used in both.
Classes
  ClassDescription
BuildLogViewerControl
This control is used to view the help file builder log file content
ContentLayoutEditorControl
This user control is used to edit content layout files
EntityReferencesControl
This user control is used to look up code entity references, code snippets, tokens, images, and table of content entries and allows them to be dragged and dropped into a topic editor window.
FileContentNeededEventArgs
This is used to contain the event arguments for the FileContentNeeded event.

This control is used to preview MAML topic files
