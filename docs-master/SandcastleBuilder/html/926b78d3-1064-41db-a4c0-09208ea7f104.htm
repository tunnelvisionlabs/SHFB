Error ARL0001: The Additional Reference Links plug-in has not been configured yet

Cause

The plug-in was added to the project but was not configured. The default configuration is not sufficient to use it in a build.

Solution

Open the editor for the PlugInConfigurations project property, select the plug-in, and click the Configure button to specify a valid configuration for it.
