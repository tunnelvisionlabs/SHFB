Remarks
This can be used by any plug-in that does not produce a reflection information file using MRefBuilder.exe. In such cases, the API filter is not applied unless the plug-in uses this method. If the reflection information file is produced by MRefBuilder.exe, there is no need to use this method as it will apply the API filter automatically to the file that it produces.
