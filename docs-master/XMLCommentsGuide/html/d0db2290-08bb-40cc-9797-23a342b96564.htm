c

This element is used to indicate that text within a description should be marked as code.

Syntax

This inline element can be used within any other element.

XML
Copy to ClipboardPrint
<c>text</c>
<c>text</c>
Remarks

Use this element to mark inline text as code. Use the code element to mark a multi-line block of lines as code.
