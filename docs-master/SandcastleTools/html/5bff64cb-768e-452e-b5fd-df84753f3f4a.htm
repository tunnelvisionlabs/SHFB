Version 2014.2.15.0 Beta

Release notes for version 2014.2.15.0 Beta.

See the Version 2014.1.26.0 Beta release notes for important information on breaking changes made to the build tools for these beta releases.

Sandcastle Tools
  • Fixed the syntax generators so that they skip the getter/setter attribute code if no getter/setter info is present (i.e. AjaxDoc reflection data files).
  • Fixed MRefBuilder so that it does not crash under certain conditions when generating extension method information.
Presentation Styles
  • Fixed a problem with the presentation style base classes so that they work properly when a base path is not set.
  • Fixed the VS2010 style's formatting of the list XML comments element so that it is consistent with the other presentation styles when term and description elements are used.
  • Modified the VS2010 style so that it generates the member list in the class topic rather than as a separate Member List topic to match MSDN.
  • Removed the unnecessary paragraph element from the auto-outline resource items.
