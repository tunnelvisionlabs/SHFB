Version 1.1.0.0

Version 1.1.0.0 was released on February 3rd, 2009.

Changes in This Release

  • Added constructors to support providerOption elements in the configuration. These are required for .NET 3.5 support.

     Important
    Be sure to update the version number to 1.1.0.0 in the compiler element in any Web.config files that use the custom code providers.
